#  PrimeThermal v1.1-OneUI
Adaptive Thermal & CPU Manager for Samsung Galaxy S10 (Exynos 9820)

## 📱 Description
Stops your S10 from overheating while keeping performance sane.
Dual-sensor logic (Skin + Core) + auto CPU frequency control.

## ✅ Features
- Real-time thermal monitoring
- Auto-switch modes: POCKET / IDLE / LIGHT / ACTIVE / CHARGING
- Battery-friendly tuning
- Detailed logging (`/data/local/tmp/s10_thermal.log`)
- Safe & reversible (no bootloop risk)

## 📥 Installation
1. Download ZIP
2. Flash via Magisk/KernelSU or TWRP
3. Reboot
